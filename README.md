# nist-80053-security-assessment-package
# Able Behavioral Health — NIST 800-53 Security Assessment Package

## Project Overview

This project simulates a full cybersecurity governance, risk, and compliance (GRC) assessment for a fictional regional healthcare and behavioral health organization, **Able Behavioral Health**.

The purpose of this project is to demonstrate practical knowledge of:

- NIST SP 800-53 Rev. 5
- HIPAA Security Rule compliance
- Risk assessment methodologies
- Security documentation
- Policy development
- Vulnerability management
- POA&M remediation tracking
- Governance and compliance operations

This project was designed as a hands-on portfolio exercise to showcase skills relevant to cybersecurity GRC analyst roles.

---

# Organization Profile

## Able Behavioral Health

Able Behavioral Health is a regional healthcare and group home provider operating across three facilities in the southeastern United States.

### Environment Overview

- 3 healthcare/group home facilities
- 150 employees
- Hybrid cloud environment
- Microsoft 365 enterprise environment
- Windows 10/11 endpoints
- Cloud-based Electronic Health Record (EHR) platform
- VPN remote access solution
- HIPAA-regulated protected health information (PHI)

The objectives of this assessment include:
